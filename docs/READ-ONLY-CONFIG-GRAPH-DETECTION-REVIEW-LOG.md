## Graph engine
- Module: src/config_graph.rs
- Root config handling: starts from a selected/root config path and records the root as the first connected file.
- Source/include support: parses `source = path`, `source=path`, `include = path`, and `include=path`.
- Relative path support: relative source/include paths resolve against the file containing the source/include line.
- Tilde path support: `~` and `~/...` resolve through the configured HOME directory.
- Symlink handling: records symlink status and target path using read-only filesystem metadata.
- Missing file handling: records unreadable/missing connected files without panicking.
- Cycle handling: tracks active and visited resolved paths and records cycles instead of recursing forever.
- Script scanning: reads configured script directories as plain text only, detects file references and mutation-looking patterns, and never executes scripts.
- Generated file hints: records likely generated files from `generated by`, `do not edit`, or `managed by` comments near the top of a connected file.
