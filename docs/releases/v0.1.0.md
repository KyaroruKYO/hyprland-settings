# Hyprland Settings v0.1.0

## Release scope

Guarded normal-scalar safe-batch Hyprland Settings app for Hyprland v0.55.2 data/model.

## What is included

- Native Rust + GTK4/libadwaita app.
- 341 readable / 341 writable / 0 blocked app model.
- Guarded safe-batch writes for eligible normal scalar settings.
- Backup-before-write behavior in the guarded write path.
- Reread verification after guarded writes.
- Recovery/restore contracts in the guarded write path.
- Source/include-aware config detection.
- Connected-file diagnostics.
- Generated/script-managed/symlink/current-profile blockers.
- Duplicate/missing/default/high-risk/display/profile blockers.
- GTK safe-env AT-SPI validation coverage.
- Desktop and AppStream metadata.

## What is intentionally not included

- Missing/default insertion.
- Duplicate auto-resolution.
- High-risk/display-render write expansion.
- Structured-family writes.
- Profile/mode switching.
- Hyprland reload/runtime mutation.
- Hyprland 0.55.4 data/model migration.

## Safety notes

- The project remains pinned to v0.55.2 data/model for this release.
- The user's runtime package may be Hyprland 0.55.4, but this release does not migrate app data to 0.55.4.
- Do not treat blocked features as bugs in this release scope.
